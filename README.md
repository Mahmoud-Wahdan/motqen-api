# Motqen Backend API

- Backend API for a role-based service marketplace connecting clients and workers.
- This service handles authentication (OTP-based), user management, job lifecycle, reviews, and authorization logic.

# Running the Project

- copy `.env.example` to `.env` and setup your configurations

## Development
```bash
npm install
npm run dev
```

# Production
```bash
npm run build
npm start
```
